DISPlay Commands

DISPlay Commands are used to set the display system. DISPlay Commands include:

  • :DISPlay:TYPE
  • :DISPlay:GRID
  • :DISPlay:PERSist
  • :DISPlay:MNUDisplay
  • :DISPlay:MNUStatus
  • :DISPlay:SCReen
  • :DISPlay:CLEar
  • :DISPlay:BRIGhtness
  • :DISPlay:INTensity
  • :DISPlay:DATA?

:DISPlay:TYPE

:DISPlay:TYPE <type>

:DISPlay:TYPE?

:DISPlay:GRID

:DISPlay:GRID <grid>

:DISPlay:GRID?

:DISPlay:PERSist

:DISPlay:PERSist {{1|ON}|{0|OFF}}

:DISPlay:PERSist?

:DISPlay:MNUDisplay

:DISPlay:MNUDisplay <time>

:DISPlay:MNUDisplay?

:DISPlay:MNUStatus

:DISPlay:MNUStatus {{1|ON}|{0|OFF}}

:DISPlay:MNUStatus?

:DISPlay:SCReen

:DISPlay:SCReen <scr>

:DISPlay:SCReen?
