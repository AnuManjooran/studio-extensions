ACQuire Commands

 

ACQuire Commands are used to set the acquisition mode for oscilloscope. ACQuire Commands include:

  • :ACQuire:TYPE
  • :ACQuire:MODE
  • :ACQuire:AVERages
  • :ACQuire:SRATe?

:ACQuire:TYPE

:ACQuire:TYPE <type>

:ACQuire:TYPE?

:ACQuire:MODE

:ACQuire:MODE <mode>

:ACQuire:MODE?

:ACQuire:AVERages

:ACQuire:AVERages <count>

:ACQuire:AVERages?

:ACQuire:SRATe?

:ACQuire:SRATe?[{<CHANnel<n>}]
