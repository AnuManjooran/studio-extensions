:CALibrate Commands

 

Command List:

  • :CALibrate:QUIT
  • :CALibrate:STARt

:CALibrate:QUIT

Syntax

:CALibrate:QUIT

Description

Exit the self-calibration at any time.

Related command

:CALibrate:STARt

:CALibrate:STARt

Syntax

:CALibrate:STARt

Description

The oscilloscope starts to execute self-calibration.

Explanation
  • The self-calibration operation can make the oscilloscope quickly reach its optimum working state to obtain the most accurate measurement values.
  • During the self-calibration, all the channels of the oscilloscope must be disconnected from the inputs.
  • The functions of most of the keys are disabled during the self-calibration. You can send the :CALibrate:QUIT command to quit the self-calibration.
